# Glossar

Dieses Glossar definiert die zentralen Begriffe rund um **Calvin**. Die hier
festgelegten Bezeichnungen gelten **global** – also einheitlich in Code,
Dokumentation, UI und Kommunikation.

| Begriff | Definition |
|---------|------------|
| **Calvin** | Name des Produkts: ein Buchungssystem für Konferenzräume (und später Arbeitsplätze) an allen INNOQ-Standorten. |
| **INNOQ-Mitarbeiter** | Die Zielgruppe und Nutzer von Calvin. Personen, die Räume für Meetings, Workshops und Zusammenarbeit buchen. |
| **Standort** | Eine der 8 INNOQ-Niederlassungen, an denen Räume buchbar sind. |
| **Konferenzraum** | Ein buchbarer Raum für Meetings und Workshops. Verfügt über Eigenschaften wie Größe und Ausstattung. |
| **Raumbuchung** | Die Reservierung eines Konferenzraums für einen bestimmten Zeitraum durch einen INNOQ-Mitarbeiter. |
| **Buchung** | Oberbegriff für eine Reservierung – aktuell eine Raumbuchung, später auch eine Arbeitsplatzbuchung. |
| **Verfügbarkeit** | Der Zustand, ob ein Raum in einem Zeitraum frei oder belegt ist. |
| **Verfügbarkeitsanzeige** | Funktion, die die Verfügbarkeit von Räumen darstellt und durch automatische Blockierung Doppelbuchungen verhindert. |
| **Doppelbuchung** | Zwei sich überschneidende Buchungen für denselben Raum im selben Zeitraum. Soll durch Calvin verhindert werden. |
| **Blockierung** | Automatisches Sperren eines bereits gebuchten Zeitraums, um Doppelbuchungen zu vermeiden. |
| **Buchungsübersicht** | Ansicht, die zeigt, welche Räume wann gebucht bzw. belegt sind. |
| **Ausstattung** | Die Einrichtung und technischen Merkmale eines Raums (z. B. Beamer, Whiteboard), relevant für die Raumwahl. |
| **Raumgröße** | Die Kapazität eines Raums, relevant für die Auswahl des passenden Raums zum Anlass. |
| **Arbeitsplatzbuchung** | Geplante spätere Funktion: Buchung von Einzelarbeitsplätzen inkl. Übersicht, wer wann im Büro ist. |
| **Arbeitsplatz** | Ein einzelner buchbarer Arbeitsplatz (Teil der späteren Ausbaustufe). |
